A Companion plugin for Blaize Theme, which provides the options for Custom Post Types and Customizer Options.

== Description ==
Blaize Companion is a companion plugin for Blaize Theme. The plugin creates three basic Custom Post Types ( Portfolio, Teams & Testimonials). These Custom Post types are the building blocks for creating the website using the Blaize theme. Additionally the plugin also provides the useful customizer options to configure the Website using the Blaize theme.

== Installation ==
This section describes how to install the plugin and get it working.

e.g.

Offline Installation
-------------------------------
1. Download a copy of Blaize Companion plugin zip file from WordPress directory.
2. Extract the plugin zip file.
3. Now Upload the plugin files to the \'/wp-content/plugins/\' directory.
4. Activate the plugin through the \'Plugins\' screen in WordPress.
5. Once Activated the Custom Post Types will be visible on the WordPress Dashboard.
6. Add the Contents to the those Custom Post Types as per your requirement.
7. The Additional Customizer Options will be visible under Dashboard > Appearance  > Customize. Configure the Options as per your requirement.

Online Installation
-------------------------------
1. Go to Dashboard > Plugins > Add New.
2. Type \'Blaize Companion\' keyword in search field.
3. Click on \'Install Now\' button and Activate the plugin
4. Once Activated the Custom Post Types will be visible on the WordPress Dashboard.
5. Add the Contents to the those Custom Post Types as per your requirement.
6. The Additional Customizer Options will be visible under Dashboard > Appearance  > Customize. Configure the Options as per your requirement.
